## Databricks magic commands
- `%fs` - File System
- `%sql` - SQL
- `%python` - Python
- `%scala` - Scala
- `%sh` - Shell
- `%md` - Markdown
- `%env` - Environment variables
- `%lsmagic` - List all magic commands
- `%run` - Run a notebook
